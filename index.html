Le but est de se préparer au projet du groupe avec les exercices de miniprojet

  • miniprojet-1 : Création du répertoire miniprojet avec 3 sous répertoires (programmes, urls et tableaux). Création d'un script pour récupérer les informations démandées des sites web
  • miniprojet-2 : Création d'un autre script pour transformer le résultat qui est une table du premier script en une table de format HTML
  • miniprojet-3 : Création d'un script pour améliorer notre site
